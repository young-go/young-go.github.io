国光的 macOS 软件清单列表

MacOS APP 清单
发布日期:   2020-07-16
更新日期:   2020-07-16

来源: 国光
文章作者: 国光
文章链接: https://www.sqlsec.com/2020/06/macsoft.html#toc-heading-79
咳咳又想白嫖文章？本文章著作权归作者所有，任何形式的转载都请注明出处。

昨天凌晨苹果 WWDC 发布会上展示了macOS 的下一代版本 macOS 11.0 Big Sur，国光我手贱升级了，变化真的巨大，状态栏、通知栏还有其他 UI 元素都大改动了，图标也全都圆角矩形了，而且更加拟物很像 macOS 10.10 之前的风格，虽然这些变动都很有新鲜感，但是国光我安装的很多软件都翻车无法使用， PD 虚拟机也挂了、brew 也无法安装软件了…..没错国光我抹掉磁盘重新安装回 macOS 10.15.5 Catalina了，每次系统安装完都需要安装一大堆软件，所以这篇文章就这样诞生了。

前言

国光是一名信息安全从业者，偶尔写写代码和剪辑，所以下面的软件清单也都是按照我的职业特性来列举的。另外还有有一些敏感的破解版资源，国光我单独放在了这个页面了：macOS精品软件

所有软件均来源于互联网分享，本页主要是为大家带来更好的下载体验，大家低调使用。

0

0xED

官网地址：0xED

0xED 是基于 Cocoa 框架的本机 OS X 十六进制编辑器，速度快，内存占用小，免费支持简体中文。

010 Editor

相关链接：MAC精品软件

专业的十六进制编辑器，CTF 比赛 MISC 杂项必备工具之一。

360 极速浏览器

官网地址：360 极速浏览器 | Mac版

因为工作性质问题，有时候需要多个浏览器来登录不同账号测试，360 的极速浏览器也还算是不错的。

360 压缩大师

官网地址：App Store - 360压缩大师

免费无广告，和 eZip 差不多的压缩软件，App Store 上的评分也还算不错，也可以考虑尝试一下。

A

aText

相关链接：MAC精品软件

通过缩写快速录入内容，文字工作者必备，有时候重度写代码的话也是一个不错的选择。

阿里旺旺

官网地址：阿里旺旺

上班网购聊天必备，电脑上操作比手机效率要高出 N 倍呐。

爱奇艺

官网地址：App Store - 爱奇艺

买前生产力，买后爱奇艺。MBP 的屏幕和外放音质都还算是不错的，不用来刷爱奇艺可惜了。

Audacity

官网地址：Mac | Audacity

音频隐写必备软件，曾在 XCTF 线下比赛遇到一个 MISC 就是使用这个软件做出来的。

B

Bartender

相关链接：MAC精品软件

当你安装足够多的软件时，这个时候菜单栏就会变得很拥挤，这个时候就需要这款方便的管理菜单栏图标软件了。

百度网盘

官网地址：百度网盘 客户端下载

基本上国光我不会分享百度网盘链接的，但是总会有人使用百度网盘分享资源，无奈。

Burp Suite

参考链接：macOS下如何优雅的使用Burp Suite

Web 从业者必备抓包改包工具，Top10 黑客工具之一。

C

Calibre

官网地址：calibre - Download for macOS

配合 Gitbook 导出 PDF 必备软件，不过软件本身作为本地的图书管理库也还算不错，就是颜值国光我不能接受。

超级右键

官网地址：App Store - 超级右键

macOS 自带的右键功能太弱了，这款神器可以完整支持右键定制，如进入终端，拷贝路径，剪切，新建文件等使用功能。

D

Dock

官网地址：Empowering App Development for Developers | Docker

Docker 搭建环境还是很方便的啦，有了 Docker 还要啥虚拟机。

钉钉

官网地址：钉钉下载

酷公司用钉钉，作为员工也只能跟着用钉钉了，还算不错的一个软件吧。

E

Expressions

相关链接：MAC精品软件

高颜值的正则表达式软件，始于颜值，终于颜值，国光我从 macOS 10.12 开始就一直在用这款正则测试 APP。

eZip

官网地址：专为 macOS 而设计的压缩软件

正如官网描述这样：专为 macOS 而设计的压缩软件，国光我主要看中了他免费的特点。

F

FileZilla

官网地址：Download FileZilla Client for Mac OS X

非常经典强大的 FTP 管理工具，免费版的功能完全足够使用，macOS 下 FTP 管理使用的不二选择。

G

Google Chrome

官网地址：Google Chrome 网络浏览器

macOS 系统自带的 Safari 浏览器太弱了，完全不能满足我们这种重度使用浏览器的用户，虽然 Chrome 比较吃内存，但是不得不承认使用体验还是相当不错的。

H

HandShaker

官网地址：Smartisan 应用下载-锤子商城

老罗以前在锤子的时候开发的良心应用，让 Android 设备和 macOS 直接拷贝传输文件很方便。

I

IINA

官网地址：IINA - The modern media player for macOS

高颜值且强大的视频播放器，关键还免费，macOS 必备软件之一。

iTerm2

官网地址：iTerm2 - macOS Terminal Replacement

正如官网介绍那样，iTerm2 是 macOS 上必不可少的终端软件之一，支持各种灵活丰富的定制，关键还免费开源。

K

Keynote

官网地址：App Store - Keynote 讲演

强的 PPT 只做软件，苹果官方出品，虽然国内用户不多，但是做的 PPT 效果的确很赞，国光曾用过这个做过一期视频的开场：垃圾佬99.9999%完美的黑苹果主机驱动分享

Kite

官网地址：Kite - AI Coding Assistant for Python and JavaScript

强大的 Python 语法代码提示，支持各种主流的编辑器，Python 开发者值得一试。

L

蓝信

官网地址：蓝信官网下载

护网行动官方指定 APP，安全度比较高，安全圈的朋友应该都用过的吧。

Little Snitch

相关链接：MAC精品软件

老牌的 Mac 防火墙软件，联网控制很方便并且还支持数据可视化展示，用来屏蔽一些破解软件更新简直完美。

Logitech G HUB

官网地址：Logitech 支持 + 下载

在高强度的工作下，鼠标还是很有必要的，国光我一直使用罗技 G302 游戏鼠标，性价比很高而且在 macOS 下驱动效果也很不错。

M

Magnet

相关链接：MAC精品软件

Windows10 自带强大的窗口分屏，但是在 macOS 分屏体验就不是那么好了，所以如果你经常打开各种各样窗口工作的话，那么你需要这款窗口整理分屏神器。

Maipo

官网地址：App Store - Maipo for 微博

macOS 上最好用的新浪微博客户端，没有乱七八糟的广告和弹窗，微博爱好者必备。

MAMP PRO

相关链接：MAC精品软件

macOS 上经典的 PHP Web 服务集成环境，类似于 Windows 下的 PHPStudy。

Microsoft Remote Desktop

相关链接：MAC精品软件

因为是微软官方出的，所以这可能是 macOS 上最简约好用的 Windows 远程桌面连接工具。

Microsoft Excel

官网地址：App Store - Microsoft Excel

国光我并不喜欢用 Excel ，但是被迫又不能不用 Excel。

Microsoft PowerPoint

官网地址：App Store - Microsoft PowerPoint

国光我并不喜欢用 PPT ，但是被迫又不能不用 PPT。

Microsoft To Do

官网地址：App Store - Microsoft To Do

微软良心 APP，每日规划立 Flag 神器，国光我也正是用了这款工具规划一点点完成了各种各样的任务，充实地度过每一天。另外欢迎加入我们的信息安装学习记录督促小组

Microsoft Word

官网地址：App Store - Microsoft Word

国光我并不喜欢用 Word ，但是被迫又不能不用 Word。

Mos

官网地址：Github - Mos

一个用于在 macOS 上鼠标平滑滚动效果的小工具, 让你的鼠标滚轮爽如触控板般丝滑。

Motrix

官网地址：一款全能的下载工具 - Motrix

迅雷实际上也不错的，但是经常有一些敏感资源被下架导致无法下载，这个时候就需要这一款下载工具了。

N

Navicat Premium

相关链接：MAC精品软件

强大的数据库管理工具，支持各种主流的数据库，颜值也很高，完美支持中文。

Nmap

官网地址：Download the Free Nmap Security Scanner for Linux/Mac/Windows

Nmap 端口扫描必备工具，图形化界面 Zenmap 官方也适配了 macOS 可直接安装，顺便也配置安装好了命令行的 Nmap ，一举两得。

P

Paragon NTFS for Mac

相关链接：MAC精品软件

macOS 读写 NTFS 磁盘利器，经常遇到同事使用 NTFS 格式的 U 盘来拷贝文件，这个时候这个软件的作用就显示出来了。

Parallels Desktop 15

相关链接：MAC精品软件

在 macOS 上的所有虚拟机软件中，国光愿称他为最强。

Paste

相关链接：MAC精品软件

macOS 剪贴板粘贴工具，可以自定义剪贴板分组，大大提高了工作效率。

PDF Expert

相关链接：MAC精品软件

实际上 macOS 自带的预览来查看 PDF 功能已经足够了，只是这个软件支持快速定位到 PDF 文件所在位置，以及添加目录也比较方便，不愧是付费软件。

PyCharm

学生或者开发者认证可以免费试用 1 年。

编写大型 Python 项目必备编译器，国光我曾用 PyCharm 写了我的 Django 博客以及 Django Web 漏扫。

Q

QQ

官网地址：QQ Mac版

macOS 下的 QQ 还是比较简约的，无广告弹窗等，而且自带的屏幕录制和文字识别功能也比较实用。

S

ScreenFlow

相关链接：MAC精品软件

macOS 强大的屏幕录制工具，支持 60 FPS 满帧的录制，并且自带简单的剪辑功能也比较实用。

ShadowsocksX-NG

官网地址：Github-ShadowsocksX-NG-R8

IT 从业人员必备软件，配合终端下的 proxychains4 或者 Chrome 的 SwitchOmega 浏览器代理插件一起食用效果最佳。

Shredo

相关链接：MAC精品软件

实用的文件粉碎机，可以帮助我们完全彻底的删除文件、目录，保护个人隐私和机密文档的安全。

SideNotes

相关链接：MAC精品软件

颜值很高的侧边栏笔记工具，完美支持 markdown，唤醒调用也比较方便，值得一试。

SnippetsLab

相关链接：MAC精品软件

优雅的代码片段管理器，对于经常写脚本的国光来说，可以很方便地管理曾经写的脚本。

搜狗输入法

官网地址：搜狗输入法Mac版

macOS 自带的输入法词库不是很懂国情，另外使用 capslock 来切换输入法的操作也反人类。所以这里国光推荐大家使用经典的搜狗输入法，在 macOS 下还是比较良心的，没有啥弹窗广告，另外斗图的功能也比较好用。

SSH Tunnel

相关链接：MAC精品软件

SSH 图形化代理管理工具，实际上直接使用命令行下代理也 OK ,但是图形化界面管理多个代理的优势就比较明显了。

Sublime Text

官网地址：Sublime Text

简约实用的代码编辑器，最新版本都已经到 Sublime Text Dev 4 的版本了，经典如初。

T

Teamviewer

官网地址：下载 Mac 版 TeamViewer，进行远程桌面访问和协作

远程控制很好用，可以经常用来远程帮助别人，完爆国内的向日葵远控。

Tecent Lemon

官网地址：腾讯柠檬清理

腾讯柠檬清理，是 CleanMyMac 的完美替代品，良心应用，Mac 必备软件之一。

Telegram

官网地址：Telegram for macOS

高匿名聊天软件，用的不多，但是不能没有这个软件。另外 macOS 的版本代理连接比较玄学，需要设置代理然后重启 TG 重复几次操作才可以成功，不过颜值真的比 TG Desktop 要强很多。

腾讯视频

官网地址：App Store - 腾讯视频

爱奇艺开了 2 年会员了，今年来试试看腾讯视频怎么样，貌似资源也挺多的。

腾讯会议

官网地址：腾讯会议 macOS 客户端

远程演示开会必备，今年疫情影响貌似用户量增长也蛮高的。

Typora

官网地址：Typora — a markdown editor, markdown reader

国光认为这是目前体验最好的 markdown 编辑器，我博客里面的绝大多数文章都是通过 Typora 编辑器书写的。

V

Visual Studio Code

官网地址：Visual Studio Code - Code Editing. Redefined

VSCode 神级编辑器，图形化做的很好，用户体验很完美，关键还免费，程序员必备编辑器。

W

微信

官方地址：微信Mac版

微信 Mac 版本的 Beta 版现在支持小程序啦，用起来更加舒服了，另外配合这个 Mac 微信小助手插件简直无敌。

Wireshark

官方地址：Wireshark · Download

专业高级的运维人员必备工具，不过国光我只用来做一些简单的 CTF 流量分析的题。

网易云音乐

官网地址：App Store - 网易云音乐

网易云音乐国光我安装的是 App Store 的版本，启动速度比之前从官网安装的要快很多，对了记得去掉菜单栏歌词显示，以节约菜单栏空间。

X

迅雷

官网地址：Mac 迅雷

迅雷下载直链速度还是很快的，虽然广告有点多，但速度真的完爆业内其他产品，真香！

Xnip

官网地址：App Store Xnip - 截图 & 标注

Xnip 是一个强大的截图软件，支持自定义比例，标注以及滚动截图，免费版足够用。

Y

优酷

官网地址：App Store - 优酷

这世界很酷，总体还算是不错的了，也是国内常用的视频播放客户端之一。
